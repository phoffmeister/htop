## Overview

HTML to PDF converter based on [headless_chrome](https://crates.io/crates/headless_chrome) crate
and inspired by [html2pdf](https://crates.io/crates/html2pdf) crate.

## Installation

```
$ cargo install htop
```

## Usage

```
$ htop --help
```
